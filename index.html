Tutorial

Menu

Click on categories ("+" icon) to expand them

Images

  • Click on the dropdown input to select the image type
  • URL : copy the direct link of the image you want (.i.e http://website.com/image.png)
  • File : click on "explore" button and select the image you want

Colors

Enter a HEX/RGB value or click on the colored rectangle to open the color picker

Font

  • Open fonts.google.com
  • Choose the font you want
  • Copy the name of the font
  • Paste it here

Styles/Mini-themes

Click on the checkboxes/radio buttons to select the style/mini-theme you want
